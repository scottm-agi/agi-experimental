# Task
Generate a SHORT title (1-3 words) for a chat conversation based on the content.

# Rules
- Output ONLY the title. Nothing else.
- 1-3 meaningful words maximum
- Focus on the main topic, person names, or subject matter
- Use proper capitalization
- Do NOT output explanations, quotes, or formatting

# Good examples
Scott's Story
Diamond Planet
Code Review
Database Setup
Image Analysis

# Bad examples (DO NOT output these)
- "Here is a title: ..."
- "Chat Naming Assistant"
- "New Chat"
- Any sentence or explanation
