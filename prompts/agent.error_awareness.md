# ⚠️ Error Awareness — 5-Why Course-Correction

Before retrying any failed operation, you MUST apply root-cause analysis:

1. **WHY** did the error occur? (identify the symptom)
2. **WHY** does that root cause exist? (dig deeper)
3. **WHAT** must change in your approach? (fix the cause, not the symptom)

**Rules:**
- DO NOT retry the same operation with the same arguments
- DO NOT guess — verify your assumptions first
- If you hit the same error twice, STOP and change strategy completely
