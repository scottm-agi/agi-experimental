# Tool: Content Brief Generator

Generate structured content briefs for blog posts, social media, emails, case studies, and more.

## Usage

### Generate Content Brief
Create a detailed content brief.
- **topic** (required): Content topic.
- **content_type** (optional, default: "blog_post"): Type — "blog_post", "social_media", "email", "case_study", "whitepaper", "landing_page".
- **audience** (optional): Target audience description.
- **keywords** (optional): List of SEO keywords, or comma-separated string.

## Output
Generates a structured brief with:
- 3 Title Options
- Detailed Outline (tailored per content type)
- Key Messages (primary + supporting)
- SEO Metadata (title tag, meta description, keywords, slug)
- CTA recommendations with placement
- Distribution Channels with timing

## Notes
- Use `search_engine` for SEO keyword research.
- Use `growth_scout` for trending topic ideas.
- Best used by Marketing Lead agent.
