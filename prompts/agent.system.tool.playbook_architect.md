# Tool: Playbook Architect

Generate structured sales and marketing playbooks based on proven templates.

## Usage

### Generate Playbook
Create a specific type of sales playbook.
- **type**: The type of playbook ("prospecting", "qualification", "closing").
- **industry**: (Optional) The target industry.
- **product**: (Optional) The product or service name.

## Notes
- This tool provides the structural framework for agents to fill in with specific research data.
- It ensures consistency across all sales enablement materials.
