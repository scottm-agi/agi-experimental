# Tool: Prospect Profiler

Generate structured prospect profiles for sales targeting.

## Usage

### Profile a Prospect
Create a comprehensive prospect profile template.
- **company** (required): Company name to profile.
- **industry** (optional): Industry sector for tailored analysis.
- **context** (optional): Additional context (e.g., "met at SaaStr conference").

## Output
Generates a structured markdown profile with:
- Company Overview (size, revenue, tech stack, recent news)
- Key Contacts table
- Pain Points mapped to value props
- Budget Signals
- Competitive Landscape
- Recommended Approach

## Notes
- The template provides the structure — use `search_engine`, `scrape_url`, and Researcher to fill in real data.
- Always use `fact_check` to verify any data claims before presenting.
- Best used by Sales Enabler and Account Leader agents.
