# Tool: Competitive Matrix

Generate formatted competitive analysis matrices with feature comparison and market positioning.

## Usage

### Build Competitive Analysis
Create a structured competitive analysis.
- **product** (required): Your product/service name.
- **competitors** (required): List of competitor names, or comma-separated string.

## Output
Generates a comprehensive competitive matrix with:
- Feature Comparison grid (checkmark table)
- Pricing Comparison table
- Per-competitor Strengths & Weaknesses
- Win/Loss Themes with mitigation strategies
- Market Positioning comparison

## Notes
- Use `search_engine` and `scrape_url` to research each competitor.
- Delegate deep research to Researcher via `call_subordinate`.
- Verify all pricing and feature claims with `fact_check`.
- Best used by Marketing Lead and Sales Enabler agents.
