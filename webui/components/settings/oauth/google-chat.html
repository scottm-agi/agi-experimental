Google Chat Setup

To enable Google Chat, you need to provide your OAuth 2.0 Client credentials from Google Cloud Console.

1
Go to Google Cloud Console

Visit the Credentials page. Select your project (or create a new one).

2
Enable Google Chat API

Make sure the "Google Chat API" is enabled in the API Library.

3
Create OAuth Client ID

Click "Create Credentials" > "OAuth client ID". Choose Desktop App as the application type.

4
Paste JSON below

Download the JSON file or copy the content and paste it into the field below.
